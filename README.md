# SPEAK_test_mock

UW–Madison SPEAK 口语考试控时训练小软件：7 套全真模拟卷 + 专项练习，带计时环、提示音、实时转写、录音回放，训练结束后可保存每题录音、导出整场模考记录（JSON）。

## 怎么运行

第一次使用：

1. 安装 [Node.js](https://nodejs.org/)（LTS 版即可，装好后自带 npm）
2. 在本项目文件夹里打开终端，执行：

```bash
npm install   # 只需第一次执行
npm run dev
```

3. 浏览器打开终端里显示的地址（一般是 http://localhost:5173），允许使用麦克风即可开始练习。

以后每次只需要 `npm run dev` 这一条命令。

## 注意

- 推荐用电脑版 **Chrome 或 Edge**，实时转文字功能只有它们支持；其他浏览器计时、录音、回放不受影响。
- 录音只保存在页面内存里，刷新就没了 —— 练完请在「训练回顾」页点「保存」下载录音，或「导出记录」保存整场数据。
